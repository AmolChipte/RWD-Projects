In this Project, API-Key is use for Currency Values, So please Follow Following Steps in Order to Use your Own API-Key -
1. Visit https://www.exchangerate-api.com/
2. Add you Correct Email Address and Click on Get Free Key
3. Verify Entered Email and Open Dashboard
4. Paste it in INDEX.JS file
NOTE: Since it is Free Plan so it have limit of 1500 Requests Per Month.
