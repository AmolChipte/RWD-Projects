In this Project, API-Key is use for Random Photos, So please Follow Following Steps in Order to Use your Own API-Key -
1. Visit https://unsplash.com/
2. Click on LOG IN
3. Create new Account if account did not exists
4. Click on Burger-Menu (Horizontal three lines) located at Right top Corner.
5. Go to Products -> Developer/APIs -> Your Apps
6. Click on NEW APPLICATION and Accept Terms and Fill Application information
7. Find Keys Block, in it Access Key is Given.
8. Copy Access Key and Paste it in INDEX.JS file
NOTE: Since it is Free Plan so it have limit of 50 Requests Per Hour.
