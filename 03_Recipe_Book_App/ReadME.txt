In this Project, API-Key is use for Random Recipes, So please Follow Following Steps in Order to Use your Own API-Key -
1. Visit https://spoonacular.com/food-api
2. Click on START NOW
3. Sign Up for Login
4. Verify Email
5. Go to MY CONSOLE if DASHBOARD Did not Appears by Default
6. Go to PROFILE and Click SHOW/HIDE API KEY
7. Copy API-KEY
8. Paste it in INDEX.JS file
NOTE: Since it is Free Plan so it have limit of 150 Requests Per Day.
