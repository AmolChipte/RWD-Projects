In this Project, API-Key is use for Weather Data of User-defined City, So please Follow Following Steps in Order to Use your Own API-Key -
1. Visit https://openweathermap.org/
2. Click on SIGN IN
3. Create new Account if account did not exists
4. Verify Email
5. Go to MY API KEYS
6. Copy API-KEY
7. Paste it in INDEX.JS file
NOTE: Since it is Free Plan so it have limit of 1000 Requests Per Day.
