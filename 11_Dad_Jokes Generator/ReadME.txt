In this Project, API-Key is use for Random Joke, So please Follow Following Steps in Order to Use your Own API-Key -
1. Visit https://api-ninjas.com/
2. Click on LOG IN
3. Create new Account if account did not exists
4. Not Mandatory - Click on API
5. Not Mandatory - Find DAD JOKES in API DIRECTORY LIST
6. Not Mandatory - You can Use Sample Request URL
7. Go to MY ACCOUNT -> API KEY -> Show API KEY.
8. Copy Access Key and Paste it in INDEX.JS file
NOTE: Since it is Free Plan so it have limit of 10000 Requests Per Month.
